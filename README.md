## Metaq是什么

Metaq是一款分布式，高性能，高可靠的消息中间件，支持严格的消息顺序，大量的消息堆积等大型互联网分布式应用特性，在淘宝及阿里巴巴子公司被广泛应用于异步解耦，数据源同步，实时计算等领域，每天处理250亿+条消息。
